# Creative Test Agent — Краткое описание для клиента

## Что делает система

Creative Test Agent — это локальный инструмент для предварительного тестирования креативных материалов (тексты объявлений, сценарии, макеты) до того, как они будут показаны клиенту.

Система оценивает креатив по заданным критериям, проверяет соответствие бренду и тону коммуникации, анализирует реакцию разных сегментов аудитории и формирует структурированный отчёт с рекомендациями.

## Почему важен закрытый контур (closed-loop)

Все данные обрабатываются локально:

- креативы не отправляются в облачные API;
- бренд-профили и данные об аудитории остаются на вашем компьютере;
- результаты тестов не покидают локальную среду.

Это особенно важно для агентств, работающих с NDА и конфиденциальными материалами клиентов.

## Что показывает демо

Демонстрация на вымышленном бренде NovaBank включает:

1. **Загрузку креативов** — текстовые варианты объявлений.
2. **Создание бренд-профиля** — тон коммуникации, ограничения.
3. **Профили аудитории** — разные сегменты с их потребностями и возражениями.
4. **Запуск анализа** — система оценивает креатив по 8 критериям.
5. **Проверку brand safety** — выявление рискованных формулировок.
6. **Сравнение вариантов (A/B)** — объективное сравнение нескольких креативов.
7. **Отчёт для клиента** — чистый, понятный HTML-отчёт без технических деталей.

## Какие материалы можно загружать

| Формат | Расширения | Примечание |
|--------|------------|------------|
| Текст | .txt, .md | Креативы, сценарии, описания |
| PDF | .pdf | Текстовые PDF (не отсканированные) |
| Изображения | .png, .jpg, .webp | Метаданные извлекаются, OCR не реализован |

## Какие отчёты формируются

- **JSON** — структурированные данные для интеграций.
- **Markdown** — читаемый текстовый отчёт.
- **HTML** — стилизованная веб-страница.
- **Client-facing HTML** — отшлифованная версия для показа клиенту (без внутренних деталей).
- **PDF stub** — подготовлен интерфейс, генерация PDF — в планах.

## Что входит в MVP

- Создание и загрузка креативов.
- Управление бренд-профилями и профилями аудитории.
- Структурированный пре-тест креативов (stub mode / локальная LLM).
- Отчёты в нескольких форматах.
- A/B сравнение вариантов.
- Локальный веб-интерфейс.
- Полностью закрытый контур обработки.

## Что можно доработать в платном пилоте

- Подключение реальной локальной LLM (Ollama, LM Studio) для более точной оценки.
- OCR для отсканированных PDF и изображений.
- Анализ видео-креативов.
- Генерация PDF-отчётов.
- Интеграция с CRM / системами управления проектами агентства.
- Адаптация критериев оценки под специфику клиента.
- Деплой в защищённой инфраструктуре клиента.
