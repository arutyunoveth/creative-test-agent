# Вопросы для подготовки пилота

## О креативах

1. Какие типы креативов нужно проверять: баннеры, тексты, PDF, презентации, лендинги?
2. Какое количество креативов планируется тестировать в месяц?
3. Есть ли готовые примеры креативов для пилота?
4. Требуется ли поддержка анимированных или видео-креативов (в настоящее время не поддерживаются)?

## О бренде и tone of voice

1. Существует ли брендбук или гайдлайн по tone of voice?
2. Есть ли список запрещённых формулировок?
3. Какие обещания и claims допустимы, а какие нет?
4. Есть ли готовые brand guidelines в машиночитаемом формате?
5. Нужно ли тестировать соответствие нескольким брендам/продуктам?

## О целевой аудитории

1. Какие сегменты аудитории нужно симулировать?
2. Есть ли портреты целевой аудитории (персоны)?
3. Какие боли, мотивации и возражения у каждого сегмента?
4. Сколько сегментов нужно для пилота?

## О форматах файлов

1. В каких форматах хранятся креативы (txt, pdf, png, jpg, webp, docx, pptx)?
2. Какой максимальный размер файлов ожидается?
3. Нужна ли поддержка пакетной загрузки?
4. Требуется ли обработка в формате Figma/Photoshop/Sketch (в настоящее время не поддерживаются)?

## О требованиях к закрытому контуру

1. Есть ли требования по хранению данных в определённой юрисдикции?
2. Требуется ли физическая изоляция данных от интернета?
3. Нужен ли аудит доступа к системе?
4. Есть ли политика безопасности, которую необходимо соблюдать?
5. Требуется ли шифрование данных на диске?

## О локальной инфраструктуре

1. Какая операционная система используется (macOS, Windows, Linux)?
2. Есть ли возможность установки дополнительного ПО (Ollama, Tesseract)?
3. Есть ли GPU для запуска локальных LLM (опционально)?
4. Какой объём RAM доступен на рабочей машине?
5. Нужно ли запускать систему на сервере или на локальном ноутбуке?

## О желаемом формате отчёта

1. Какой формат отчёта предпочтителен: Markdown, HTML, PDF?
2. Нужна ли возможность экспорта в DOCX?
3. Какие разделы должны быть в отчёте обязательно?
4. Кто будет читать отчёт: креативная команда, аккаунт-менеджеры, клиент?
5. Нужна ли русскоязычная версия отчёта?

## О критериях успеха пилота

1. Как мы поймём, что пилот прошёл успешно?
2. Какие метрики будем отслеживать?
3. Сколько итераций тестирования планируется в рамках пилота?
4. Кто будет принимать решение о продлении или расширении пилота?
5. Какой бюджет заложен на пилот?
